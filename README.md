# Interactive DSA Calendar (2026)

A modern, responsive calendar web application designed to help users practice Data Structures & Algorithms (DSA) daily. Each day is mapped to a curated problem, with support for notes, range selection, and progress tracking.

---

## Features

### Calendar UI

- Clean **wall calendar aesthetic**
- Monthly navigation (restricted to year **2026**)
- Fully responsive (desktop + mobile)

### Day Selection

- Select **single date** to view "Problem of the Day"
- **Future dates are locked **

### DSA Integration

- Each month is mapped to a **specific Data Structure**
- Daily problems linked from **LeetCode**
- Structured learning roadmap for the entire year

### Notes System

- Add notes for each day
- Notes are saved in **localStorage**
- Automatically loads when date is selected

### Visual Indicators

- Highlight **today’s date**
- Highlight **selected date**
- Highlight range selection
- Indicator for saved notes

---
